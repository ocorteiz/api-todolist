## 💻 Sobre o projeto

To-Do-List é uma API REST para cadastro de tarefas usando Spring Boot e conceitos de validação e teste.

---

## ⚙️ Funcionalidades

- [x] Cadastro de tarefas
- [x] Listagem de tarefas
- [x] Atualização de tarefas
- [x] Exclusão de tarefas

---
